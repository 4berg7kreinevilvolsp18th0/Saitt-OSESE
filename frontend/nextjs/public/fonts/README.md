# Шрифты брендбука ОСС

## Размещение шрифтов

Поместите следующие файлы шрифтов в эту папку (`public/fonts/`):

### SF UI Text
- `SFUITEXT-BOLD.TTF`
- `SFUITEXT-SEMIBOLD.TTF`
- `SFUITEXT-LIGHT.TTF`
- `SFUITEXT-MEDIUM.TTF`
- `SFUITEXT-REGULAR.TTF` (если есть)

### SF UI Display
- `SFUIDISPLAY-HEAVY.TTF`
- `SFUIDISPLAY-SEMIBOLD.TTF`
- `SFUIDISPLAY-BOLD.TTF`
- `SFUIDISPLAY-MEDIUM.TTF`
- `SFUIDISPLAY-ULTRALIGHT.TTF`
- `SFUIDISPLAY-THIN.TTF`
- `SFUIDISPLAY-REGULAR.TTF` (если есть)

## Использование

Шрифты автоматически подключены в `globals.css` и доступны через CSS классы:
- `font-sf-text` - для основного текста (SF UI Text)
- `font-sf-display` - для заголовков (SF UI Display)

## Примечание

Если у вас есть файлы шрифтов, скопируйте их в эту папку. Шрифты будут автоматически загружены браузером.
