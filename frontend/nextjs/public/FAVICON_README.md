# Favicon

Текущий файл `favicon.ico` повреждён (всего 13 байт).

## Решение

Замените `favicon.ico` на валидный ICO файл:

1. **Вариант 1:** Создайте новый favicon.ico (16x16 или 32x32 пикселей)
2. **Вариант 2:** Используйте один из существующих логотипов:
   - `Лого вектор красное.png` - можно конвертировать в ICO
   - `Лого вектор белое.png` - можно конвертировать в ICO

## Онлайн конвертеры

- https://convertio.co/ru/png-ico/
- https://www.favicon-generator.org/

## Размеры

Рекомендуемые размеры для favicon:
- 16x16 пикселей (основной)
- 32x32 пикселей (для Retina дисплеев)
- 48x48 пикселей (для Windows)
