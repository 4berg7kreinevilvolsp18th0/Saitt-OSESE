# Инструкция по добавлению логотипа

## Куда поместить файлы логотипа

Поместите файлы логотипа в папку `frontend/nextjs/public/`:

- `logo.png` - основной логотип в формате PNG (рекомендуется)
- `logo.svg` - альтернативный вариант в формате SVG

## Рекомендации по формату

### PNG логотип:
- Размер: минимум 512x512 пикселей (для высокого качества)
- Формат: PNG с прозрачным фоном
- Цвет: белый или красный (#D11F2A) на прозрачном фоне

### SVG логотип:
- Векторный формат для масштабирования без потери качества
- Оптимизированный файл

## Как это работает

Компонент Logo автоматически:
1. Сначала пытается загрузить `logo.png`
2. Если PNG не найден, пробует `logo.svg`
3. Если ни один файл не найден, показывает SVG fallback (встроенный логотип)

## Где используется логотип

- **Header** (верхняя навигация) - размер 40px с подписью "ОБЪЕДИНЕННЫЙ СОВЕТ СТУДЕНТОВ"
- **Главная страница** - размер 120px в центре страницы

## Проверка

После добавления файлов:
1. Перезапустите dev-сервер (`npm run dev`)
2. Проверьте отображение логотипа в header и на главной странице
3. Убедитесь, что подпись "ОБЪЕДИНЕННЫЙ СОВЕТ СТУДЕНТОВ" отображается корректно
