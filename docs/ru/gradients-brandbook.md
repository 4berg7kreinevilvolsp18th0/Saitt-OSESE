# Градиенты брендбука ОСС для комитетов

## Обзор

Каждый комитет/направление имеет свой уникальный градиент, соответствующий брендбуку ОСС. Градиенты взяты из официальных презентаций и материалов ОСС ДВФУ.

## Градиенты для каждого направления

### 1. Правовой комитет (Legal)
- **Цвет:** `#1F2A44` (тёмно-синий)
- **Градиент:** `linear-gradient(135deg, #1F2A44 0%, #2A3F5F 50%, #1F2A44 100%)`
- **Светлая тема:** `linear-gradient(135deg, #E8EBF0 0%, #D1D9E6 50%, #E8EBF0 100%)`
- **Источник:** Брендбук ОСС

### 2. Инфраструктурный блок (Infrastructure)
- **Цвет:** `#2A7FFF` (голубой, светлый)
- **Градиент:** `linear-gradient(135deg, #2A7FFF 0%, #4A9FFF 50%, #2A7FFF 100%)`
- **Светлая тема:** `linear-gradient(135deg, #DBEAFE 0%, #BFDBFE 50%, #DBEAFE 100%)`
- **Источник:** Брендбук ОСС - светлый голубой градиент

### 3. Стипендиальный комитет (Scholarship)
- **Цвет:** `#2E8B57` (зелёный, светлый)
- **Градиент:** `linear-gradient(135deg, #2E8B57 0%, #3EAB67 50%, #2E8B57 100%)`
- **Светлая тема:** `linear-gradient(135deg, #D1FAE5 0%, #A7F3D0 50%, #D1FAE5 100%)`
- **Источник:** Брендбук ОСС - светлый зелёный градиент

### 4. Иностранным студентам (International)
- **Цвет:** Коричнево-оранжевый (из презентации)
- **Градиент:** `linear-gradient(135deg, #5C3A1A 0%, #8B5A2B 50%, #CD853F 100%)`
- **Светлая тема:** `linear-gradient(135deg, #FFF4E6 0%, #FFE4B5 50%, #FFF4E6 100%)`
- **Источник:** Презентация "Иностранный блок ОСС" - коричнево-оранжевый градиент (от тёмно-коричневого к оранжевому)

### 5. Другое / FAQ (Neutral)
- **Цвет:** Тёмно-красный (из презентации)
- **Градиент:** `linear-gradient(135deg, #6B0000 0%, #8B0000 50%, #B91C1C 100%)`
- **Светлая тема:** `linear-gradient(135deg, #FEE2E2 0%, #FECACA 50%, #FEE2E2 100%)`
- **Источник:** Презентация "Общее" - тёмно-красный градиент

## Использование в коде

### В компонентах

```tsx
import { gradientBg, gradientBorder } from '../lib/theme';

// Применение градиента
<div className={`${gradientBg(direction.colorKey)} ${gradientBorder(direction.colorKey)}`}>
  {/* Контент */}
</div>
```

### Tailwind классы

Градиенты доступны через Tailwind классы:

- `bg-gradient-legal`
- `bg-gradient-infrastructure`
- `bg-gradient-scholarship`
- `bg-gradient-international`
- `bg-gradient-neutral`

Для светлой темы автоматически применяются варианты с суффиксом `-light`.

## Где применяются градиенты

1. **Карточки направлений** (`DirectionCard`)
   - Фон карточки с градиентом
   - Граница с цветом направления
   - Декоративные элементы при hover

2. **Страницы направлений** (`/directions/[slug]`)
   - Заголовочный блок с градиентом
   - Карточки кейсов с градиентным фоном
   - Декоративные размытые элементы

3. **Другие элементы**
   - Badge компоненты (опционально)
   - Акцентные элементы

## Эффекты

### Hover эффекты
- Увеличение масштаба (`scale-[1.02]`)
- Усиление тени (`shadow-lg`)
- Изменение прозрачности градиента
- Градиентные акценты при наведении

### Анимации
- Плавные переходы (`transition-all duration-300`)
- Декоративные размытые элементы
- Градиентные акценты с blur эффектом

## Настройка

Градиенты настраиваются в:
- `tailwind.config.js` - определение градиентов
- `lib/theme.ts` - функции для применения градиентов
- `app/globals.css` - поддержка светлой темы

## Соответствие брендбуку

Все градиенты взяты из официальных презентаций ОСС ДВФУ:
- ✅ Используются цвета из реальных презентаций
- ✅ Сохраняется читаемость текста
- ✅ Поддержка светлой и тёмной темы
- ✅ Единый стиль для всех направлений
- ✅ Соответствие визуальному стилю ОСС

## Источники градиентов

1. **Инфраструктурный блок** - из презентации с тёмно-синим фоном
2. **Стипендиальный комитет** - из презентации с тёмно-зелёным фоном
3. **Иностранный блок** - из презентации с коричнево-оранжевым градиентом
4. **Общее** - из презентации с тёмно-красным градиентом
5. **Правовой комитет** - из основного брендбука ОСС
