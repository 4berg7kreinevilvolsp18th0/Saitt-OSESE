# Быстрый старт для дизайнера

Чеклист для начала работы над дизайном сайта ОСС ДВФУ.

## Первые шаги

### 1. Изучи проект (30 минут)
- [ ] Открой текущую версию сайта в браузере
- [ ] Прочитай `docs/ru/designer-integration.md`
- [ ] Изучи `docs/ru/professional-design-system.md`
- [ ] Посмотри брендбук ОСС

### 2. Настрой Figma (15 минут)
- [ ] Установи плагины:
  - Figma Tokens
  - Figma to Code
  - Figma Dev Mode
- [ ] Создай файл "ОСС ДВФУ Design System"
- [ ] Настрой структуру папок (см. `designer-integration.md`)

### 3. Создай дизайн-систему (1-2 часа)
- [ ] Добавь все цвета из палитры в Color Styles
- [ ] Создай Text Styles для всех размеров
- [ ] Настрой Effect Styles для теней
- [ ] Создай базовые компоненты

### 4. Начни проектировать
- [ ] Создай макеты страниц
- [ ] Добавь интерактивные прототипы
- [ ] Документируй спецификации

## Чеклист перед передачей

### Обязательно
- [ ] Все цвета из палитры добавлены
- [ ] Типографика соответствует системе
- [ ] Компоненты созданы с variants
- [ ] Responsive версии для всех breakpoints
- [ ] Темная и светлая тема
- [ ] Все состояния компонентов (hover, active)
- [ ] Комментарии в Figma добавлены
- [ ] Дизайн-токены экспортированы

### Желательно
- [ ] Интерактивный прототип
- [ ] Презентация дизайна
- [ ] Анимации и переходы
- [ ] Примеры использования компонентов

## Быстрые команды

### Экспорт токенов из Figma
1. Открой плагин "Figma Tokens"
2. Выбери все токены
3. Экспорт → JSON
4. Отправь разработчику

### Создание компонента
1. Создай frame
2. Добавь Auto Layout
3. Создай variants (Theme, State)
4. Добавь Component Properties
5. Экспортируй спецификации

Удачи! Если что-то непонятно — спрашивай.
