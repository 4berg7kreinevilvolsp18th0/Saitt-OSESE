# Статистика по школам ДВФУ

## Обзор

Система автоматически группирует обращения по школам и институтам ДВФУ на основе поля `institute` в таблице `appeals`.

## Поддерживаемые школы

- **ИМО** - Институт Мирового Океана
- **ПИ** - Политехнический Институт
- **ПИШ** - Передовая Инженерная Школа «Институт Биотехнологий, Биоинженерии и Пищевых Систем»
- **ЮШ** - Юридическая Школа
- **Шминж** - Школа Медицины и Наук о Жизни
- **ИФКИС** - Институт Физической Культуры и Спорта
- **ИМКТ** - Институт Математики и Компьютерных Технологий
- **ИНТПМ** - Институт Наукоемких Технологий и Передовых Материалов
- **ВИ** - Восточный Институт
- **ШИГН** - Школа Искусств и Гуманитарных Наук
- **ШП** - Школа Педагогики
- **ШэМ** - Школа Экономики и Менеджмента

## Использование

### В админ-панели

1. Перейдите в **Дашборды** → **Статистика по школам**
2. Выберите период (7 дней, 30 дней, 90 дней, все время)
3. Просмотрите:
   - Общую статистику по всем школам
   - График распределения обращений
   - Круговую диаграмму
   - Детальную таблицу по каждой школе
4. Кликните на школу в таблице для просмотра деталей

### Через API

```bash
# Получить статистику по всем школам
GET /api/analytics/schools?start_date=2024-01-01&end_date=2024-12-31

# Получить статистику по конкретной школе
GET /api/analytics/schools?school_code=ИМО
```

## Нормализация названий

Система автоматически нормализует названия институтов:
- Ищет точное совпадение по коду (ИМО, ПИ, и т.д.)
- Ищет частичное совпадение в полном названии
- Если не найдено, сохраняет оригинальное название

## Рекомендации

Для корректной статистики:
1. При создании обращения указывайте полное название института или его код
2. Используйте стандартные названия из списка школ
3. Проверяйте статистику регулярно для выявления несоответствий
