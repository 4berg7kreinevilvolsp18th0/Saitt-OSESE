# Профессиональная система дизайна ОСС ДВФУ

## Обзор

Система дизайна создана для отражения **элегантности, строгости и официальности** ОСС как серьезной организации, при этом сохраняя **заботу и поддержку студентов**.

## Ключевые принципы

### 1. Строгость и официальность
- Четкая типографическая иерархия
- Профессиональные шрифты (SF UI Display/Text)
- Минималистичный, но выразительный дизайн
- Четкие границы и структурированность

### 2. Элегантность
- Плавные, профессиональные анимации
- Тонкие тени и эффекты глубины
- Сбалансированные пропорции
- Премиальные визуальные эффекты

### 3. Забота о студентах
- Дружелюбные, но профессиональные цвета
- Понятная навигация
- Мягкие переходы и анимации
- Доступность и удобство использования

## Анимации и переходы

### Профессиональные easing функции
```css
--ease-out-expo: cubic-bezier(0.19, 1, 0.22, 1)
--ease-in-out-quart: cubic-bezier(0.76, 0, 0.24, 1)
--ease-out-quart: cubic-bezier(0.25, 1, 0.5, 1)
--ease-in-out-cubic: cubic-bezier(0.65, 0, 0.35, 1)
```

### Типы анимаций

1. **pageFadeIn** - Плавное появление страницы
2. **fadeInUp** - Появление снизу вверх
3. **fadeInDown** - Появление сверху вниз
4. **slideInRight/Left** - Появление с боков
5. **scaleIn** - Масштабирование при появлении

### Каскадные анимации
- Задержки: 100ms, 200ms, 300ms, 400ms, 500ms
- Создают эффект последовательного появления элементов

## Компоненты

### Professional Card
Элегантные карточки с:
- Четкими границами
- Плавными тенями
- Hover-эффектами с подъемом
- Backdrop blur для глубины

### Professional Button
Строгие кнопки с:
- Градиентными фонами
- Shimmer-эффектом при hover
- Многослойными тенями
- Плавными переходами

**Варианты:**
- Primary - основной акцент (красный градиент)
- Secondary - вторичный (прозрачный с границей)

### Elegant Link
Ссылки с:
- Подчеркиванием при hover
- Плавными переходами цветов
- Фокус-кольцами для доступности

## Цветовая система

### Основные цвета
- **OSS Red**: #D11F2A - основной акцент
- **Dark**: #0F1115 - фон темной темы
- **Light Background**: Градиент от белого к серому

### Направления
- **Legal**: Темно-синий (#1F2A44) + Золотистый (#D4AF37)
- **Infrastructure**: Синий (#2A7FFF)
- **Scholarship**: Зеленый (#2E8B57)
- **International**: Желтый (#F5B301)
- **Neutral**: Серый (#6B7280)

## Типографика

### Заголовки (SF UI Display)
- **h1**: 700 weight, letter-spacing: -0.03em
- **h2**: 600 weight, letter-spacing: -0.02em
- **h3**: 600 weight, letter-spacing: -0.02em

### Текст (SF UI Text)
- Основной: 400-500 weight
- Полужирный: 600 weight
- Жирный: 700 weight

## Утилиты

### Классы для анимаций
- `.animate-page-enter` - появление страницы
- `.animate-fade-in-up` - появление снизу
- `.animate-fade-in-down` - появление сверху
- `.animate-slide-in-right/left` - появление с боков
- `.animate-scale-in` - масштабирование

### Классы для стилей
- `.professional-card` - элегантная карточка
- `.professional-button` - строгая кнопка
- `.elegant-link` - элегантная ссылка
- `.hover-lift` - подъем при hover
- `.focus-ring` - фокус для доступности
- `.smooth-transition` - плавные переходы

## Доступность

- Фокус-кольца для всех интерактивных элементов
- Плавные переходы для снижения нагрузки на глаза
- Четкая типографическая иерархия
- Достаточный контраст цветов

## Производительность

- Оптимизированные easing функции
- CSS-анимации вместо JavaScript
- Минимальное использование blur-эффектов
- Эффективные transitions

## Использование

### Пример карточки
```tsx
<div className="professional-card hover-lift animate-fade-in-up">
  {/* Контент */}
</div>
```

### Пример кнопки
```tsx
<button className="professional-button professional-button-primary focus-ring">
  Действие
</button>
```

### Пример ссылки
```tsx
<Link href="/path" className="elegant-link focus-ring">
  Текст ссылки
</Link>
```
