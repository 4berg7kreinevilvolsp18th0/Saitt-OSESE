# Фоны направлений и декоративные элементы

## Обзор

Каждое направление ОСС имеет уникальный фон и декоративные размытые элементы, создающие визуальную атмосферу страницы.

## Цвета фонов

### Правовой комитет (Legal)
- **Основной фон**: `#1D1E25` (тёмно-серый с фиолетовым оттенком)
- **Размытые элементы 1**: `#383B72` (фиолетово-синий)
- **Размытые элементы 2**: `#3B3F70` (фиолетово-синий, более тёмный)

### Инфраструктурный блок (Infrastructure)
- **Основной фон**: `#241E34` (тёмно-фиолетовый)
- **Размытые элементы 1**: `#1E6BAA` (голубой)
- **Размытые элементы 2**: `#1C809C` (бирюзовый)

### Стипендиальный комитет (Scholarship)
- **Основной фон**: `#251D1D` (тёмно-красный)
- **Размытые элементы 1**: `#1C9C56` (зелёный)
- **Размытые элементы 2**: `#1C9C65` (зелёный, более светлый)

### Иностранный блок (International)
- **Основной фон**: `#2B2121` (тёмно-коричневый)
- **Размытые элементы 1**: `#EDAE26` (оранжевый)
- **Размытые элементы 2**: `#EB9D2F` (оранжевый, более тёмный)

### Объединение (Union/Neutral)
- **Основной фон**: `#241E34` (тёмно-фиолетовый)
- **Размытые элементы 1**: `#C240FF` (маджентовый)
- **Размытые элементы 2**: `#D026D9` (маджентовый, более светлый)

## Декоративные элементы

Каждое направление использует два набора размытых декоративных элементов (Group 1 и Group 2), которые создают глубину и атмосферу:

### Group 1
- **Ellipse 5**: Большой элемент слева внизу (opacity: 0.2, blur: 600px)
- **Ellipse 1**: Большой элемент справа вверху (opacity: 0.7, blur: 400px)
- **Ellipse 3**: Средний элемент справа вверху (opacity: 0.6, blur: 150px)
- **Ellipse 4**: Маленький элемент справа вверху (opacity: 1.0, blur: 100px)

### Group 2
- **Ellipse 6**: Большой элемент справа вверху (opacity: 0.2, blur: 600px)
- **Ellipse 1**: Большой элемент слева внизу (opacity: 0.7, blur: 400px)
- **Ellipse 3**: Средний элемент слева внизу (opacity: 0.6, blur: 150px)
- **Ellipse 4**: Маленький элемент слева внизу (opacity: 1.0, blur: 100px)

## Техническая реализация

### CSS классы

Цвета размытых элементов определены в `globals.css`:
- `.bg-blur-legal-1`, `.bg-blur-legal-2`
- `.bg-blur-infrastructure-1`, `.bg-blur-infrastructure-2`
- `.bg-blur-scholarship-1`, `.bg-blur-scholarship-2`
- `.bg-blur-international-1`, `.bg-blur-international-2`
- `.bg-blur-neutral-1`, `.bg-blur-neutral-2`

### Tailwind конфигурация

Основные цвета фонов определены в `tailwind.config.js`:
- `legal`: `#1D1E25`
- `infrastructure`: `#241E34`
- `scholarship`: `#251D1D`
- `international`: `#2B2121`
- `neutral`: `#241E34`

### Использование в компонентах

Функции из `lib/theme.ts`:
- `getBlurColor1(colorKey)` - возвращает класс для первого набора размытых элементов
- `getBlurColor2(colorKey)` - возвращает класс для второго набора размытых элементов

Пример использования:
```tsx
const blurColor1 = getBlurColor1(direction.colorKey);
const blurColor2 = getBlurColor2(direction.colorKey);
```

## Адаптивность

Декоративные элементы адаптируются под разные размеры экранов:
- **Мобильные устройства**: Уменьшенные размеры элементов (250-450px)
- **Планшеты**: Средние размеры элементов (400-700px)
- **Десктоп**: Полные размеры элементов (800-1258px)

Позиционирование также адаптируется для оптимального отображения на всех устройствах.

## Светлая тема

В светлой теме фоны направлений заменяются на светлые градиенты:
- Правовой комитет: `#E8EBF0`
- Инфраструктурный блок: `#DBEAFE`
- Стипендиальный комитет: `#D1FAE5`
- Иностранный блок: `#FEF3C7`
- Объединение: `#F3F4F6`

Декоративные размытые элементы в светлой теме имеют пониженную прозрачность для более мягкого эффекта.
