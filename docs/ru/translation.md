# Переводы сайта

Как работать с переводами и добавлять новые языки на сайте ОСС ДВФУ.

## Где хранятся переводы

Все переводы находятся в папке `frontend/nextjs/locales/`:
- `ru.json` — русский язык (основной)
- `en.json` — английский язык

## Как это работает

В коде используется система переводов через React Context:
- `LocaleProvider` — компонент, который предоставляет переводы всему приложению
- `useLocale()` — хук для доступа к переводам в компонентах
- `t(key)` — функция для получения перевода по ключу

### Пример использования

```typescript
'use client';

import { useLocale } from '../components/LocaleProvider';

export default function MyComponent() {
  const { t } = useLocale();
  
  return (
    <div>
      <h1>{t('home.title')}</h1>
      <p>{t('home.subtitle')}</p>
    </div>
  );
}
```

## Формат файлов переводов

Файлы организованы по разделам сайта:

```json
{
  "common": {
    "loading": "Загрузка...",
    "error": "Ошибка"
  },
  "navigation": {
    "home": "Главная",
    "directions": "Направления"
  },
  "appeal": {
    "title": "Подать обращение",
    "form": {
      "title": "Тема обращения",
      "description": "Описание"
    }
  }
}
```

## Как добавить новый перевод

1. Откройте `frontend/nextjs/locales/ru.json`
2. Добавьте новый ключ в нужный раздел
3. Откройте `frontend/nextjs/locales/en.json`
4. Добавьте тот же ключ с английским переводом
5. Используйте в коде через `t('section.key')`

**Важно:**
- Не меняйте названия ключей — они должны совпадать в обоих файлах
- Переводите только значения (текст после двоеточия)
- Если в тексте есть `{{name}}` или подобные символы — оставьте их без изменений

### Пример

**Русский (ru.json):**
```json
{
  "appeal": {
    "form": {
      "title": "Тема обращения",
      "titlePlaceholder": "Кратко опишите суть вопроса"
    }
  }
}
```

**Английский (en.json):**
```json
{
  "appeal": {
    "form": {
      "title": "Appeal Title",
      "titlePlaceholder": "Briefly describe the issue"
    }
  }
}
```

## Параметры в переводах

Если нужно подставить значение, используйте `{{paramName}}`:

```json
{
  "appeal": {
    "success": {
      "message": "Ваше обращение #{{id}} принято"
    }
  }
}
```

В коде:
```typescript
t('appeal.success.message', { id: '12345' })
// Результат: "Ваше обращение #12345 принято"
```

## Основные разделы переводов

- **common** — общие фразы (кнопки, статусы, действия)
- **navigation** — пункты меню
- **home** — главная страница
- **directions** — страница направлений
- **appeal** — форма обращения
- **status** — проверка статуса
- **admin** — админ-панель
- **statistics, content, documents, contacts** — разделы сайта
- **footer** — футер

## Часто используемые фразы

## Проверка переводов

После добавления переводов:

1. Убедитесь, что структура ключей совпадает в обоих файлах
2. Проверьте, что все ключи из `ru.json` есть в `en.json`
3. Проверьте работу переключения языка на сайте
4. Убедитесь, что нет синтаксических ошибок в JSON
