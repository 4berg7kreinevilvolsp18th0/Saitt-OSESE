# Документация на русском языке

Добро пожаловать в документацию сайта ОСС ДВФУ на русском языке.

## Начните здесь

**[Начало работы](getting-started.md)**  
Настройка проекта на вашем компьютере и запуск локально. Занимает около 15 минут.

## Основные руководства

**[Деплой](deployment.md)**  
Размещение сайта в интернете с помощью Vercel. Пошаговые инструкции.

**[Настройка базы данных](database.md)**  
Настройка Supabase (база данных). Как создать таблицы, добавить пользователей, настроить роли.

**[Редактирование контента](content-editing.md)**  
Как добавлять и редактировать новости, гайды и другой контент. Написано для нетехнических пользователей.

**[Решение проблем](troubleshooting.md)**  
Частые проблемы и способы их решения. Ошибки 404, проблемы с базой данных, проблемы со сборкой.

**[Функции сайта](features.md)**  
Обзор функций сайта: тёмная/светлая тема, интеграция с Telegram, логотип и т.д.

**[Модули](modules-implementation.md)**

- Первая настройка → [Начало работы](getting-started.md)

- Добавить контент → [Редактирование контента](content-editing.md)

**Возникли проблемы?**

1. Сначала проверьте [Решение проблем](troubleshooting.md)
2. Посмотрите конкретное руководство для вашей задачи
3. Проверьте логи Vercel/Supabase
4. Проверьте переменные окружения
