## Настройка переменных окружения

В `.env.local` и Vercel добавьте:

```env
NEXT_PUBLIC_TELEGRAM_CHANNEL=oss_dvfu
NEXT_PUBLIC_TELEGRAM_API_URL=/api/telegram/posts
```

## Получение реальных постов

### Вариант 1: Telegram Bot API (рекомендуется)

1. Создайте бота через [@BotFather](https://t.me/BotFather)
2. Получите токен бота
3. Добавьте бота в канал @oss_dvfu как администратора
4. Добавьте токен в Vercel (только на сервере, не NEXT_PUBLIC_!):
   ```
   TELEGRAM_BOT_TOKEN=your-bot-token
   ```

**Ограничение:** Telegram Bot API не позволяет напрямую получать сообщения из каналов. Нужно использовать:
- Webhook для получения новых сообщений
- Или сохранять сообщения в базу данных при публикации

### Вариант 2: RSS Feed

Если канал публичный, можно использовать RSS:
- URL: `https://rss.app/v1/channels/oss_dvfu/feed`
- Требует парсинг XML/RSS

### Вариант 3: Сторонние сервисы

Используйте сервисы типа:
- Telegram RSS Proxy
- Telegram Channel Reader API
- Другие публичные API для Telegram каналов
