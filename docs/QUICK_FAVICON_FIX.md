# Быстрое исправление ошибки favicon.ico 404

## Проблема

Браузер показывает ошибку:
```
/favicon.ico:1 Failed to load resource: the server responded with a status of 404 ()
```

## Причина

Файл `frontend/nextjs/public/favicon.ico` содержит только текст "placeholder" (13 байт), а не валидный ICO файл.

## Решение (выберите один вариант)

### Вариант 1: Создать новый favicon.ico (рекомендуется)

1. Создайте изображение 32x32 пикселя с логотипом ОСС ДВФУ
2. Конвертируйте в ICO формат:
   - Онлайн: https://convertio.co/ru/png-ico/
   - Или используйте любой графический редактор
3. Сохраните как `frontend/nextjs/public/favicon.ico`
4. Закоммитьте и запушьте изменения

### Вариант 2: Использовать существующий логотип

1. Откройте `frontend/nextjs/public/Лого вектор красное.png`
2. Уменьшите до 32x32 пикселей
3. Конвертируйте в ICO: https://convertio.co/ru/png-ico/
4. Сохраните как `frontend/nextjs/public/favicon.ico`

### Вариант 3: Временно игнорировать (не критично)

Ошибка favicon не влияет на работу сайта - это только предупреждение браузера. Сайт будет работать нормально, просто не будет иконки во вкладке.

## Проверка

После создания правильного favicon.ico:

1. Очистите кеш браузера (Ctrl+Shift+Delete)
2. Откройте сайт в режиме инкогнито
3. Проверьте, что ошибка исчезла
4. Проверьте, что иконка отображается во вкладке браузера
