# Светлая тема и интеграция с Telegram

## ✅ Что добавлено

### 1. Светлая тема

- ✅ Переключатель темы в Header (иконка солнца/луны)
- ✅ Автоматическое сохранение выбранной темы в localStorage

- ✅ Компонент `TelegramPosts` для отображения постов из Telegram канала
- ✅ Отображается на главной странице в боковой панели
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Ссылки на Telegram канал

**Где отображается:**
- Главная страница (правая колонка рядом с новостями)

## Настройка Telegram интеграции

### Шаг 1: Добавьте переменные окружения

В `.env.local` и Vercel:

```env
NEXT_PUBLIC_TELEGRAM_CHANNEL=@oss_dvfu
NEXT_PUBLIC_TELEGRAM_API_URL=https://your-api.com/api
```

### Шаг 2: Создайте API endpoint (опционально)

Для получения реальных постов из Telegram создайте API route:
`app/api/telegram/posts/route.ts`

Подробная инструкция: `docs/TELEGRAM_INTEGRATION.md`

## Стили для светлой темы

Все компоненты автоматически поддерживают светлую тему через Tailwind классы:
- `dark:` - стили для темной темы
- `light:` - стили для светлой темы

Пример:
```tsx
<div className="bg-oss-dark dark:bg-oss-dark light:bg-gray-50 text-white dark:text-white light:text-gray-900">
```

## Компоненты

### ThemeToggle
Переключатель темы в Header.

### ThemeProvider
Провайдер для инициализации темы при загрузке страницы.

### TelegramPosts
Компонент для отображения постов из Telegram канала.

**Параметры:**
- `channel?: string` - имя канала (по умолчанию из env)
- `limit?: number` - количество постов (по умолчанию 5)

## Технические детали

### Хранение темы
Тема сохраняется в `localStorage` с ключом `theme`.

### Инициализация
Тема инициализируется в `ThemeProvider` при загрузке страницы.

### CSS классы
- `.dark` - применяется к `<html>` для темной темы
- `.light` - применяется к `<html>` для светлой темы
