# Инструкция для переводчиков

## Что нужно сделать

Перевести содержимое файла `frontend/nextjs/locales/en.json` с русского на английский язык.

## Где находится файл

```
frontend/nextjs/locales/en.json
```

## Как работать

1. Откройте файл `en.json` в любом текстовом редакторе
2. Файл содержит структуру JSON с переводами
3. **ВАЖНО:** Не меняйте структуру файла (названия ключей в кавычках)
4. Переводите только значения (текст после двоеточия)

## Пример

**НЕПРАВИЛЬНО** (меняете ключ):
```json
{
  "home": {
    "title": "Welcome"  ← НЕ МЕНЯЙТЕ "title"!
  }
}
```

**ПРАВИЛЬНО** (переводите только значение):
```json
{
  "home": {
    "title": "Welcome to OSS DVFU"  ← Переводите только текст здесь
  }
}
```

## Структура файла

Файл организован по разделам сайта:
- `common` - общие фразы (кнопки, статусы)
- `navigation` - пункты меню
- `home` - главная страница
- `directions` - направления работы
- `appeal` - форма обращения
- `status` - проверка статуса
- `admin` - админ-панель
- и другие разделы

## Специальные символы

Если видите `{{name}}` или подобное - **НЕ УДАЛЯЙТЕ** эти символы! Это параметры, которые подставляются автоматически.

Пример:
```json
{
  "appeal": {
    "success": {
      "message": "Your appeal #{{id}} has been accepted"
    }
  }
}
```

## После перевода

1. Сохраните файл
2. Проверьте, что JSON валидный (нет синтаксических ошибок)
3. Отправьте обновленный файл обратно

## Проверка JSON

Убедитесь, что:
- Все открывающие `{` имеют закрывающие `}`
- Все открывающие `[` имеют закрывающие `]`
- Все строки в кавычках
- Запятые между элементами на месте
- Нет лишних запятых в конце
