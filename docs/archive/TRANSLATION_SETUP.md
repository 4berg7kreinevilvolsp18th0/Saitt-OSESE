# Настройка системы переводов

## Структура файлов

Система переводов использует JSON файлы в папке `frontend/nextjs/locales/`:
- `ru.json` - русский язык (основной)
- `en.json` - английский язык

## Как это работает

1. **LocaleProvider** - React Context, который предоставляет переводы всему приложению
2. **useLocale()** - хук для доступа к переводам в компонентах
3. **LanguageToggle** - компонент переключения языка в Header
4. **t(key)** - функция для получения перевода по ключу

## Использование в компонентах

```typescript
'use client';

import { useLocale } from '../components/LocaleProvider';

export default function MyComponent() {
  const { t } = useLocale();
  
  return (
    <div>
      <h1>{t('home.title')}</h1>
      <p>{t('home.subtitle')}</p>
    </div>
  );
}
```

## Добавление новых переводов

1. Откройте `frontend/nextjs/locales/ru.json`
2. Добавьте новый ключ в нужный раздел
3. Откройте `frontend/nextjs/locales/en.json`
4. Добавьте тот же ключ с английским переводом
5. Используйте в коде через `t('section.key')`

## Что нужно перевести

Все тексты в `en.json` уже содержат базовые переводы, но их нужно проверить и улучшить профессиональными переводчиками.

Основные разделы для перевода:
- Навигация
- Формы обращений
- Статусы обращений
- Админ-панель
- Общие фразы
