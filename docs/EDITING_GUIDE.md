# Руководство по редактированию контента

Простое руководство для неопытных пользователей по редактированию материалов на сайте ОСС ДВФУ.

## Как войти в систему редактирования

1. Откройте сайт ОСС ДВФУ
2. Перейдите на страницу `/admin/login`
3. Введите ваш email и пароль (полученные от администратора)
4. После входа вы попадёте в админ-панель

## Создание нового материала

1. В админ-панели нажмите **"Контент"**
2. Нажмите кнопку **"+ Создать материал"**
3. Заполните форму:

### Тип материала
Выберите один из вариантов:
- **Новость** - для новостей и объявлений
- **Гайд** - для инструкций и руководств
- **FAQ** - для часто задаваемых вопросов

### Статус
- **Черновик** - материал сохранён, но не виден на сайте
- **Опубликовано** - материал виден всем посетителям сайта
- **Архив** - материал скрыт, но сохранён

### Направление (опционально)
Выберите направление, к которому относится материал, или оставьте "Не указано"

### Заголовок
Введите название материала. Например: "Как получить стипендию"

**Важно:** Заголовок должен быть понятным и информативным.

### Slug (URL)
Это адрес страницы в интернете. Автоматически создаётся из заголовка, но можно изменить вручную.

**Пример:** Если заголовок "Как получить стипендию", slug будет "kak-poluchit-stipendiyu"

**Правила:**
- Только латинские буквы, цифры и дефисы
- Без пробелов
- Без специальных символов

### Содержание
Введите текст материала. Поддерживается форматирование Markdown.

## Форматирование текста (Markdown)

### Заголовки
```
# Заголовок 1 (самый крупный)
## Заголовок 2
### Заголовок 3
```

### Жирный текст
```
**Этот текст будет жирным**
```

### Курсив
```
*Этот текст будет курсивом*
```

### Списки
Ненумерованный список:
```
- Пункт 1
- Пункт 2
- Пункт 3
```

Нумерованный список:
```
1. Первый пункт
2. Второй пункт
3. Третий пункт
```

### Ссылки
```
[Текст ссылки](https://адрес-сайта.ru)
```

### Пример готового текста

```
# Заголовок новости

Это первый абзац текста. Он содержит основную информацию.

## Подзаголовок

Второй абзац с **важной информацией** и *дополнительными деталями*.

### Что нужно сделать:

- Шаг 1
- Шаг 2
- Шаг 3

Подробнее читайте [здесь](https://example.com).
```

## Редактирование существующего материала

1. Перейдите в раздел **"Контент"**
2. Найдите нужный материал в списке
3. Нажмите кнопку **"Редактировать"**
4. Внесите изменения
5. Нажмите **"Сохранить"**

## Публикация и снятие с публикации

### Опубликовать материал
1. Откройте материал для редактирования
2. Измените статус на **"Опубликовано"**
3. Нажмите **"Сохранить"**

Материал сразу появится на сайте.

### Снять с публикации
1. В списке материалов найдите нужный
2. Нажмите кнопку **"Снять с публикации"**

Материал исчезнет с сайта, но останется в системе.

## Полезные советы

1. **Всегда проверяйте материал перед публикацией**
   - Прочитайте текст на ошибки
   - Проверьте ссылки
   - Убедитесь, что форматирование правильное

2. **Используйте черновики**
   - Сохраняйте незавершённые материалы как черновики
   - Публикуйте только готовые материалы

3. **Правильные заголовки**
   - Заголовок должен быть информативным
   - Избегайте слишком длинных заголовков
   - Используйте заголовки разных уровней для структуры

4. **Проверка после публикации**
   - После публикации откройте материал на сайте
   - Убедитесь, что всё отображается правильно
   - Проверьте, что ссылки работают

## Частые вопросы

**В: Как удалить материал?**
О: Измените статус на "Архив". Материал скроется с сайта, но останется в системе.

**В: Можно ли редактировать опубликованный материал?**
О: Да, просто откройте его для редактирования и сохраните изменения.

**В: Что делать, если забыл пароль?**
О: Обратитесь к администратору системы.

**В: Как добавить изображение?**
О: Пока изображения добавляются через ссылки. Загрузите изображение на внешний хостинг и вставьте ссылку в текст.

## Поддержка

При возникновении проблем:
1. Проверьте, что вы вошли в систему
2. Убедитесь, что у вас есть права на редактирование контента
3. Обратитесь к администратору системы
